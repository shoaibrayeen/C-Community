# An Array is given and every element is appearing twice except two elements. You need to find those two elements.

# Constraints
-  1 < size Of Array <= 10^6
-  0 < Array Elements <= 10^9

# Input
- Array
- Its Size

# Output
- Both Non-Repeated Elements

```
Input :  [1, 2, 3, 1, 4 , 3 ]
Output : 2 and 4

Input :  [1 , 3, 2, 3]
Output : 1 and 2
```
