# Given an unsorted array, count all distinct elements in it.

# Constraints
-  1 < size Of Array <= 10^6
-  0 < Array Elements <= 10^9

# Input
- Array
- Its Size

# Output
- return the count

```
Input :   arr[] = {10, 20, 20, 10, 30, 10}
Output : 3

Input :   arr[] = {10, 20, 20, 10, 20}
Output : 2

```
