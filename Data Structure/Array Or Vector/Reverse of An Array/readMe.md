# An Array is given and You need to reverse the original array.

# Constraints
-  1 < size Of Array <= 10^6
-  0 < Array Elements <= 10^9

# Input
- Array
- Its Size

# Output
- Reverse Array

```
Input :  [1, 34, 3, 98, 9, 76, 45, 4, 12, 121]
Output : [121, 12, 4, 45, 76, 9, 98,3, 34, 1]

Input :  [12, 121]
Output : [121, 12]
```
