## A positive number N is given to you and You need to calculate Nth Number in Tribonacci Series.
## Tribonacci Series is 0, 0, 1, 1, 2, 4, 7, 13, 24, 44, 81, ...

### Input
- N ( It should be positive)

### Output
- Return Nth Tribonacci Number


```
  N = 5
  Output = 2
  
  N = 10
  Output = 44
```
